Oscar Awards Dashboard
**Интерактивная панель управления для анализа данных о премии Оскар с 1928 по 2025 год.**

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://oscar-dashboard-dwncctcmvmxskgobhncedp.streamlit.app)

[**Живое приложение доступно здесь**](https://oscar-dashboard-dwncctcmvmxskgobhncedp.streamlit.app)

### Локальный запуск
Если хотите запустить локально:
```bash
git clone https://github.com/yuliagavrilova1111/oscar-dashboard.git
cd oscar-dashboard
pip install -r requirements.txt
streamlit run app.py
```

## Возможности
### **Аналитика и визуализация**
- **Основные тренды** — динамика номинаций и побед по годам
- **Сравнение категорий** — тепловые карты и анализ конкурентности
- **Анализ по периодам** — десятилетиям и историческим эпохам
- **Актеры и режиссеры** — статистика, гендерный анализ, рекорды
- **Студии** — сравнение Золотого века Голливуда и современного периода

### **Интерактивные возможности**
- Фильтрация данных по годам, категориям, фильмам, актерам, режиссерам, студиям
- Сохранение состояния фильтров между сессиями
- Экспорт отфильтрованных данных в CSV
- Автоматический сброс фильтров
- 6 интерактивных вкладок с различными визуализациями

### Дополнительные функции
- Прогнозная модель на основе Random Forest
- Настройка параметров ML модели
- Визуализация результатов машинного обучения

## Быстрый старт

### Предварительные требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/yuliagavrilova1111/oscar-dashboard.git
cd oscar-dashboard

# 2. Создайте виртуальное окружение
python -m venv venv

# 3. Активируйте виртуальное окружение
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Запустите приложение
streamlit run app.py
```

Приложение будет доступно по адресу: http://localhost:8501

## Технологический стек
- Frontend: Streamlit
- Визуализация: Plotly, Plotly Express
- Анализ данных: Pandas, NumPy
- Машинное обучение: Scikit-learn, imbalanced-learn
- Управление состоянием: Streamlit Session State
- Версионирование: Git, GitHub

## Данные
Данные содержат информацию о номинациях на премию Оскар с 1929 года по настоящее время и включают:
- Год церемонии (1928-2025)
- Категория награды
- Фильм
- Актеры, режиссеры, студии
- Результат (победа/номинация)

## Использование приложения
1. Настройте фильтры в левой панели:
- Диапазон лет
- Категория награды
- Название фильма
- Имя актера/режиссера
- Студия
2. Исследуйте данные через 6 вкладок:
-Основные тренды - ключевые метрики и динамика по годам
-Сравнение категорий - тепловые карты и конкурентность
-Анализ по периодам - десятилетия и исторические эпохи
-Актеры и режиссеры - статистика и гендерный анализ
-Студии - успешность студий в разные периоды
-Прогнозный анализ - ML модель для предсказания побед
3. Взаимодействуйте с данными:
- Наводите курсор на графики для детальной информации
- Используйте кнопку "Сбросить фильтры" для возврата к настройкам по умолчанию
- Экспортируйте отфильтрованные данные кнопкой "Экспортировать отфильтрованные данные"
